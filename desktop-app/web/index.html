Extension Setup

The browser extension is required for website blocking to work. Install it in each browser you use.

  1. Open your browser's extension settings
  2. Enable "Developer mode"
  3. Load the extension from: ~/.local/share/website-blocker/extension
  4. Enable the extension in incognito/private mode
